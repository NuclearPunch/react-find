 ### 리액트 환경설정
 #### 준비물  
 * NodeJS
    windows 일 경우 공식홈페이지에서 다운로드 후 설치  
    mac OR linux 일 경우

    ~~~ 
    $ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash
    ~~~

 * yarn
    npm 보다 속도가 빠르기 때문에 사용권장 공식사이트 이용   

 * editor
    VSCode 나 atom , webStorm 추천

#### 환경설정

//create-react-app 설치
